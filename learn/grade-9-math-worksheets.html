What your child will learn — Algebra I: Polynomials & Quadratics

  • To multiply (x + 2)(x + 5), use FOIL — First, Outer, Inner, Last: x² + 5x + 2x + 10 = x² + 7x + 10.
  • Factoring reverses FOIL: x² + 7x + 10 = (x + 2)(x + 5). Find two numbers that MULTIPLY to 10 and ADD to 7.
  • Zero Product Property: if (x + 2)(x + 5) = 0, then x = −2 or x = −5. Setting factors to zero solves the equation!
  • Quadratic graphs are parabolas — the U-shaped arc of every thrown ball.

Sample worksheet questions

1. Expand: (x + 5)(x + 6) = ____
Answer: x² + 11x + 30
2. Factor: x² + 11x + 24 = ____
Answer: (x + 3)(x + 8)
3. Solve: (x − 8)(x + 8) = 0
Answer: x = 8 or x = −8
4. Expand: (x + 2)(x + 2) = ____
Answer: x² + 4x + 4
5. Factor: x² + 13x + 42 = ____
Answer: (x + 7)(x + 6)
